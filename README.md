# Financial Valuation Models

This repository contains my independent financial modeling projects completed as part of my professional development in investment banking and equity research. Each model applies core valuation methodologies used in industry, including discounted cash flow (DCF) analysis and comparable company (comps) analysis.

## Included Company Valuations

### NVIDIA (NVDA)
- DCF valuation based on a 5-year FCFF projection
- Comparable company analysis using semiconductor and AI hardware peers

### Robinhood (HOOD)
- Comparable company analysis reflecting fintech sector dynamics

## Tools & Methods
- Excel modeling with dynamic assumptions
- WACC and terminal value estimation
- Multiple-based valuation and peer benchmarking


## Purpose
To demonstrate practical financial analysis skills and strong analytical reasoning, while improving proficiency in valuation techniques and business fundamentals.
